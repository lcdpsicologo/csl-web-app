# Horarios y Funcionarios

Carpeta para cargar informacion operativa del colegio:

- `horarios-profesores.csv`: bloques por docente o funcionario.
- `horarios-cursos.csv`: bloques por curso.
- `nomina-funcionarios.csv`: listado maestro de funcionarios, roles y correos.

Objetivo del modulo: saber donde deberia estar cada persona del colegio en cada bloque horario y cruzarlo con curso, sala, asignatura, rol y correo institucional.

Mantener una fila por bloque horario. Si un profesor tiene 8 bloques en la semana, debe tener 8 filas.
